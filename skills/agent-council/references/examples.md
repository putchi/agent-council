# Examples

## Tech

```bash
bash scripts/run-council.sh --council tech "Should we migrate our monolith to microservices?"
bash scripts/run-council.sh --council tech "Should we use Kafka or SQS for our event streaming layer?"
bash scripts/run-council.sh --council tech "We need to choose between PostgreSQL and MongoDB for a new service"
```
